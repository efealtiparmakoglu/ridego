# 🚗 RideGo

Araç çağırma platformu. Uber/BiTaksi tarzı.

## Özellikler
- 📍 Gerçek zamanlı konum takibi
- 🚗 Sürücü-eşleştirme algoritması
- 💳 Ödeme entegrasyonu
- ⭐ Sürücü değerlendirme
- 📊 Sürücü paneli
- 🔔 Bildirim sistemi

## Teknolojiler
- React Native
- Node.js + Express
- MongoDB + Redis
- Socket.io
- Google Maps API
